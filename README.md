# 🎲 Electronic Dice using Arduino

## 📌 Description
This project is an **Electronic Dice** built using Arduino Uno. It simulates a real dice by generating a **random number between 1 and 6** when a push button is pressed. The result is displayed on a **7-segment display**, and a **buzzer sound** provides feedback for each roll.

---

## 🔧 Features
- 🎲 Generates random numbers (1–6)
- 🔢 Displays output on 7-segment display
- 🔘 Push button to roll the dice
- 🔊 Buzzer indication on each roll
- 🔁 Prevents consecutive repetition of the same number

---

## 🧰 Components Used
- Arduino Uno  
- 7-Segment Display  
- Push Button  
- Buzzer  
- Resistors  
- Breadboard  
- Jumper Wires  

---

## ⚙️ Working Principle
1. Press the push button to roll the dice  
2. Arduino generates a random number between 1 and 6  
3. The number is displayed on the 7-segment display  
4. A buzzer sound indicates the dice roll  
5. If the same number repeats, the system automatically re-rolls
